# PR001: 实现工作台的 PR 显示组件

**状态**: Open
**分类**: Feature, UI

## 背景

为 workhub 系统添加 PR 显示组件，允许用户查看和管理 Pull Request。

## 变更内容

- 添加 PRDisplay.tsx 组件用于显示单个 PR
- 添加 PRList.tsx 组件用于显示 PR 列表
- 支持按状态、时间等条件筛选和排序
- 添加状态徽章（Open, Closed, Merged, Draft）
